# Audio Assets for Cross-Platform Racer

This directory contains the audio assets used in the Cross-Platform Racer game. 

## Audio Files

- **Background Music**: Looping tracks that enhance the racing experience.
- **Sound Effects**: Includes sounds for engine revs, collisions, and other in-game actions.

## Usage

Ensure that all audio files are properly loaded in the `PreloadScene.ts` before starting the game. Use the audio manager to control playback, volume, and other audio settings.

## Formats

Supported audio formats include:
- MP3
- WAV
- OGG

Make sure to optimize audio files for performance and compatibility across different platforms.
